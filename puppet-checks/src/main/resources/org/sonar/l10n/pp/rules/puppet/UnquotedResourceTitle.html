All resource titles must be quoted. If you are using an array of titles you must quote each title in the array, but should not quote the array itself.
Note that if the title is just a variable, it doesn't need to be quoted.

Noncompliant Code Example

file { foo:       # Noncompliant
  ensure => directory,
}

file { "${bar}":  # Noncompliant
  ensure => directory,
}

file { [
    abc,          # Noncompliant
    ${var},
  ]:
  ensure => directory,
}

Compliant Solution

file { 'foo':
  ensure => directory,
}

file { ${bar}:
  ensure => directory,
}

file { [
    'abc',
    ${var},
  ]:
  ensure => directory,
}
