Shared coding conventions allow teams to collaborate efficiently. This rule checks that all variables match the following regular expression: ^(::)?([a-z][a-z0-9_]*::)*[a-z][a-z0-9_]*$

This naming convention is compliant with the more strict Puppet 4 naming convention.

Noncompliant Code Example

file { $myFile:    # Noncompliant
  ensure => file,
  mode   => $Mode,  # Noncompliant
}

Compliant Solution

file { $my_file:
  ensure => file,
  mode   => $mode,
}
